# Rosé Recipes - Aesthetic Kitchen

A beautiful, aesthetic recipe website built with React and Tailwind CSS.

## 🚀 Quick Start

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/ENGENE282/Recipes-for-u-.git
cd Recipes-for-u-
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## 🛠️ Build

To build for production:
```bash
npm run build
```

## 🎨 Features

- Beautiful, aesthetic design with warm color palette
- Responsive layout (mobile, tablet, desktop)
- Recipe cards with images
- Category browsing
- Search functionality (ready to implement)
- Smooth hover effects and transitions
- Modern typography and spacing

## 🎯 Customization

Edit the `AestheticRecipeWebsite.jsx` component to:
- Add more recipes
- Modify categories
- Change colors (update hex values)
- Replace images with your own
- Add navigation links
- Implement search and filtering
